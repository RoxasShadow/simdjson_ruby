# simdjson gem

Experimental binding for [simdjson](https://github.com/lemire/simdjson).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'simdjson'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install simdjson
